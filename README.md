# 戦国武将ベースボール探偵団

小学校2年生向けの、算数と思考力・読解力を楽しく伸ばすWebアプリです。  
見た目は「野球 × 戦国武将 × ナゾトキ」のゲームですが、問題の裏側には中学受験算数・国語につながる単元タグを持たせています。

## 技術構成

- Next.js
- TypeScript
- Tailwind CSS
- JSONによる問題管理
- LocalStorageによる回答履歴・カード・球場・子どもプロフィール保存
- Netlifyデプロイ想定
- Supabase、ログイン、外部APIは未使用

## ローカル起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## Netlifyデプロイ方法

1. GitHubにこのプロジェクトを置きます。
2. Netlifyで「Add new site」→「Import an existing project」を選びます。
3. GitHub連携で、このリポジトリを選びます。
4. Build command は `npm run build` にします。
5. Publish directory は `.next` を使います。
6. Next.js Runtime が有効になっていることを確認します。

このリポジトリには `netlify.toml` を置いているため、Netlify側で基本設定を読み取れる構成です。

## 問題データの追加方法

問題は `src/data/questions.json` に追加します。  
各問題には、答えだけでなく以下を必ず設定します。

- `examUnit`
- `thinkingType`
- `examRelevance`
- `abstractionLevel`
- `thinkingChoices`
- `bestThinking`
- `hints`
- `parentTip`

通常ミッションは10問、短いミッション・復習・単元別ミッションは5問を基本にしています。
出題は、球場テーマ・最近出た問題・復習候補を見ながら選ばれます。

## 問題データの著作権方針

このアプリの問題は、すべてオリジナル問題として作成します。  
市販問題集、塾教材、入試過去問、無料プリントサイトの問題文・図・選択肢・解説はコピーしません。  
既存教材は単元や出題形式の参考にとどめます。

## 中学受験単元との対応方針

現在は小2向けの「半抽象問題」を中心にしています。  
将来的には同じ単元を、テーマ問題、半抽象問題、受験形式問題の3段階で出せる設計にします。

例:

- テーマ問題: ゲームやカードの文脈で楽しく考える
- 半抽象問題: A・B・Cのカードがくり返し並ぶ
- 受験形式問題: ある規則にしたがって図形や数が並ぶ
