# 次回以降の開発メモ

## 大きな方向性

「戦国武将ベースボール探偵団」を、学習ミッションを解きながら武将カードと球場を開拓していくカードゲーム風アプリへ育てる。

## やりたいこと

### 1. 武将カードに個別イラストを入れる

- 各武将カードに、その武将らしいイラストを追加する。
- 小2向けなので怖くしない。
- 甲冑・兜・旗・色・シルエットなどで違いを出す。
- 合戦・武器・暴力表現ではなく、野球チーム風の武将カードにする。
- 未所持カードはシルエットや「？」で隠す。
- 所持済みカードはカラーイラスト、レア度、能力値を表示する。

### 2. 城球場の開拓システム

- 「大坂城球場」「安土城球場」「仙台城球場」「上田城球場」「春日山城球場」「甲府館球場」などをステージ化する。
- 各球場にミッションを用意する。
- 球場をクリアすると次の球場が開放される。
- 球場ごとに出やすいカードや問題ジャンルを変える。

例:

- 仙台城球場: 伊達政宗・片倉小十郎系カードが出やすい
- 上田城球場: 真田幸村・真田昌幸系カードが出やすい
- 甲府館球場: 武田信玄・山県昌景系カードが出やすい
- 春日山城球場: 上杉謙信・直江兼続系カードが出やすい

### 3. 学習単元と球場の対応

- 球場ごとに受験単元のテーマを持たせる。

例:

- 大坂城球場: 文章題・情報整理
- 安土城球場: 規則性・周期算
- 仙台城球場: 条件整理
- 上田城球場: 場合の数・書き出し
- 春日山城球場: 読解・理由探し
- 甲府館球場: 図形・陣形

### 4. クリア条件

- 1球場につき数試合クリアで開拓完了。
- ヒントなしクリア、考え方正解、連続正解などでボーナス。
- 開拓済み球場はマップで見えるようにする。

### 5. 優先順位

1. 球場データ構造を作る。完了
2. 現在の1日3問ミッションを「球場ミッション」に対応させる。完了
3. クリア済み球場をLocalStorageに保存する。完了
4. カード排出を球場に紐づける。完了
5. 武将カードの個別イラストを追加する。仮実装済み、今後は生成画像に差し替え

## 2026-06-16 実装メモ

- `src/lib/stadiums.ts` を追加。
- `/stadiums` に城球場ロード画面を追加。
- 球場は、大坂城球場、安土城球場、仙台城球場、上田城球場、春日山城球場、甲府館球場の6つ。
- 各球場に、学習テーマ、開拓に必要な試合数、出やすい武将カード、出題キーワードを持たせた。
- ミッションに `stadiumId` を保存するようにした。
- 回答履歴にも `stadiumId` を保存するようにした。
- 結果画面で3問クリアすると、該当球場の開拓ポイントが進む。
- 1球場2試合クリアで次の球場が開放される。
- 報酬カードは、現在の球場にゆかりのある武将から出やすい。
